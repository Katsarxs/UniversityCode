# Digital Library Server
A client-server application that simulates a library management system using **Java sockets** for communication.

## Technologies Used
![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Swing](https://img.shields.io/badge/Swing-GUI-blue?style=for-the-badge&logo=java&logoColor=white)
## Features
Entry Type : *Movie* or *TV Series*.
- Searching for a entry via name or category.
- Adding an entry to the database.
- Edit an entry's properties.
- Delete an entry from the databse.
- Save current data to file.
- Load current data from file.
